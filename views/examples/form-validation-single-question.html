Server-side validation

When an error occurs:

  • add a 5px red left border to the field with the error
  • the error message be red and bold and appear above the question
  • the error message must sit inside the <label>
  • keyboard focus must be moved to the input

Also consider using the aria-invalid attribute to programmatically indicate that a field has an error. The value of the aria-invalid attribute will vary depending on the type of error.

For pages with content above the question

If there is lots of content above the question on the page, it may be useful to also show a summary box at the top of the page. with a link to correct the error.
